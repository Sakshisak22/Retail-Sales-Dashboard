# Retail Sales Analysis and Dashboard

This project focuses on analyzing retail sales data and building an interactive analytics dashboard.

The project was developed as part of a Data Analytics internship to demonstrate data analysis, visualization, and dashboard creation skills.

## Tools and Technologies
- Python
- Google Colab
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib

## Project Components

1. Data Analysis Notebook  
   A Google Colab notebook that performs:
   - Data cleaning
   - Exploratory Data Analysis (EDA)
   - Statistical insights
   - Data visualizations

2. Interactive Dashboard  
   A Streamlit dashboard that provides:
   - Sales trends
   - Profit analysis
   - Category distribution
   - Correlation analysis
   - Forecast visualization
   - Interactive charts

## Features
- Interactive visualizations
- KPI metrics for sales, orders, and profit
- Multiple chart types (line, bar, pie, scatter, heatmap)
- Sales forecasting
- Dashboard filters
